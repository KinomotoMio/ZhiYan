# Slidev Generation Contract

- Always return a full deck, not partial fragments.
- Use global frontmatter once at the top of the file.
- Separate slides with `---`.
- Prefer native Slidev layouts and markdown structures when possible.
- Select one stable visual direction for the whole deck.
- Review the final deck for malformed frontmatter, repeated weak bullet pages, and missing closing slides.
