# Recipe: Closing Takeaway

Use for the final page or a concluding synthesis page.

## Structure Signals

- One decisive closing statement.
- One compact takeaway, action, or memorable line.
- Optional badge / pill / highlighted sentence that feels quotable.

## Good Use Cases

- Final thesis.
- Call to action.
- Memorable summary or next-step statement.

## Avoid

- Reopening detailed analysis.
- Ending on weak bullets or another agenda-like list.
