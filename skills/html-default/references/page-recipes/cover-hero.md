# Recipe: Cover Hero

Use for the opening page or a major narrative reset.

## Structure Signals

- One dominant title block.
- One short positioning line or subtitle.
- One strong visual field: image wash, gradient scene, or emblematic graphic.
- Minimal supporting elements.

## Good Use Cases

- Title / thesis / event framing.
- Product launch opener.
- Talk theme and stance.

## Avoid

- Lists, multi-card comparisons, or long context paragraphs.
- Dense logos / metadata clusters that compete with the title.
