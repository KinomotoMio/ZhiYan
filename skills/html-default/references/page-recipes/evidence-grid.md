# Recipe: Evidence Grid

Use when a page needs proof, reasoning, or grouped supporting points.

## Structure Signals

- One clear takeaway title.
- 2-4 evidence blocks with visible grouping.
- Each block should contain a short title plus compact supporting text, number, or icon.
- The page should still read as one thesis supported by multiple proofs.

## Good Use Cases

- Why this matters.
- Metric / insight / proof clusters.
- Compare a few reasons, outcomes, or implications.

## Avoid

- More than four dense blocks.
- Large tables or raw paragraphs with no focal takeaway.
