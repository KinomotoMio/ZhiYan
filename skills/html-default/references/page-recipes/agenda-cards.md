# Recipe: Agenda Cards

Use for directory / agenda / chapter overview pages.

## Structure Signals

- Clear agenda title.
- 3-5 chapter cards or steps with concise labels.
- Each item should be scannable in under a second.
- Grid or vertical stack is fine, but items must feel like chapter markers, not body content.

## Good Use Cases

- Section roadmap.
- What the audience will see next.
- Framing a narrative arc before details.

## Avoid

- Turning each card into a paragraph.
- Mixing agenda with deep evidence or explanation.
